### Running server
In the application root directory run the following script.
```shell
./run.sh
```

### Running tests
In the `src` directory run the following command.
```shell
python -m unittest discover -s tests -v
```

### Python Lint
In the root directory run the following command.
```shell
pylint src/app/
```
